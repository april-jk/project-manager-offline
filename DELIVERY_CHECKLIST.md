## 🚀 快速开始

### 1. 安装依赖
```bash
cd project-manager-offline
pnpm install
```

### 2. 启动开发服务器
```bash
pnpm dev
```

### 3. 构建生产版本
```bash
pnpm build
```

### 4. 部署
参考 `docs/INSTALLATION.md` 中的部署指南。
