## 项目管理

### 创建项目

1. 点击顶部右侧的 **"+ 新项目"** 按钮
2. 在弹出对话框中填写项目信息：
   - **项目名称**: 必填，如 "我的博客"、"电商平台"
   - **项目描述**: 可选，如 "个人技术博客"
   - **项目颜色**: 选择一个颜色标签便于识别
3. 点击 **"创建"** 按钮

### 编辑项目

1. 在项目卡片上方点击 **编辑按钮** (✏️)
2. 修改项目信息
3. 点击 **"保存"** 按钮

### 删除项目

1. 在项目卡片上方点击 **删除按钮** (🗑️)
2. 确认删除（此操作不可撤销）

### 项目统计

每个项目卡片显示以下统计信息：
- **网站数**: 该项目下的网站资源数量
- **API 数**: 该项目下的 API 资源数量
- **备忘数**: 该项目下的备忘录数量

## 资源管理

### 资源类型

Project Hub 支持两种资源类型：

#### 1. 网站资源 (🌐)

用于存储网站链接和相关账户信息。

**包含字段**:
- **网站名称**: 如 "GitHub"、"阿里云"
- **URL**: 网站链接，如 `https://github.com`
- **用户名**: 可选，登录账户
- **密码**: 可选，登录密码（加密存储）
- **描述**: 可选，网站说明
- **标签**: 可选，如 "开发"、"工具"、"社区"

#### 2. API 资源 (🔑)

用于存储 API 密钥和端点信息。

**包含字段**:
- **API 名称**: 如 "OpenAI API"、"Stripe"
- **API Key**: API 密钥（加密存储）
- **API Secret**: API 密钥（可选，加密存储）
- **API 端点**: API 地址，如 `https://api.openai.com`
- **描述**: 可选，API 说明
- **标签**: 可选，如 "AI"、"支付"

### 添加资源

1. 在项目卡片下方点击 **"+ 添加资源"** 按钮
2. 选择资源类型：
   - **网站**: 用于网站链接和账户
   - **API**: 用于 API 密钥
3. 填写资源信息
4. 点击 **"保存"** 按钮

### 查看资源详情

1. 点击资源卡片
2. 预览对话框打开，显示资源详情
3. 右上角按钮：
   - **访问按钮** (🔗): 仅网站资源，直接打开网站
   - **编辑按钮** (✏️): 进入编辑模式

### 编辑资源

1. 点击资源卡片
2. 在预览对话框中点击 **编辑按钮** (✏️)
3. 修改资源信息
4. 点击 **"保存"** 按钮

### 删除资源

1. 点击资源卡片
2. 在预览对话框中点击 **删除按钮** (🗑️)
3. 确认删除

### 密码和密钥安全

- **隐藏显示**: 密码和 API Key 默认显示为 ••••••••
- **点击查看**: 点击密码字段可切换显示/隐藏
- **加密存储**: 所有密码和 API Key 使用 AES-256-GCM 加密
- **本地存储**: 数据完全存储在浏览器，不上传到服务器

## 数据管理

### 导出数据

导出所有项目和资源为 JSON 文件，便于备份或迁移。

1. 点击顶部菜单按钮 (⋮)
2. 选择 **"导出数据"**
3. 选择导出格式：
   - **JSON**: 标准 JSON 格式
   - **CSV**: 表格格式（仅资源）
4. 文件将自动下载

### 导入数据

从之前导出的文件恢复数据。

1. 点击顶部菜单按钮 (⋮)
2. 选择 **"导入数据"**
3. 选择要导入的文件
4. 确认导入（现有数据将被覆盖）

### 清空数据

清除所有本地存储的数据。

⚠️ **警告**: 此操作不可撤销，请先导出数据作为备份。

1. 点击顶部菜单按钮 (⋮)
2. 选择 **"清空所有数据"**
3. 确认清空

## 安全性

### 数据加密

- **加密算法**: AES-256-GCM（军用级加密）
- **密钥派生**: PBKDF2（100,000 次迭代）
- **加密字段**: 密码、API Key、API Secret
- **加密强度**: 256 位密钥

### 本地存储

- **存储位置**: 浏览器 LocalStorage
- **数据量**: 通常 5-10MB（浏览器限制）
- **持久性**: 浏览器关闭后数据仍保留
- **隐私**: 数据不上传到任何服务器

### 隐私保护

✅ **完全离线**: 无需网络连接，所有操作在本地完成  
✅ **无账户**: 无需创建账户，无用户追踪  
✅ **无广告**: 完全无广告，无第三方追踪  
✅ **数据所有权**: 您完全拥有所有数据  

### 安全建议

1. **定期备份**: 定期导出数据作为备份
2. **浏览器安全**: 保持浏览器和操作系统最新
3. **隐私浏览**: 避免在公共计算机上使用
4. **密码强度**: 使用强密码保护重要账户
5. **清除缓存**: 在公共计算机上使用后清除浏览器缓存

   ├─ 官网 (公司官方网站)

### 资源标签

使用统一的标签便于分类和搜索：

- **按功能**: 前端、后端、数据库、API
- **按平台**: AWS、阿里云、Azure、Google Cloud
- **按用途**: 开发、测试、生产、备份
- **按类型**: 工具、文档、账户、密钥

### 命名规范

- **项目名称**: 使用清晰的项目标识，如 "电商平台-v2"
- **资源名称**: 包含平台和用途，如 "GitHub-个人账户"、"OpenAI-API"
- **描述**: 简明扼要，包含关键信息

### 数据备份策略

1. **定期备份**: 每周导出一次数据
2. **多地备份**: 保存在本地、云盘、移动设备
3. **版本管理**: 标记备份日期，如 "backup-2026-01-25.json"
4. **加密备份**: 将备份文件放在加密文件夹中

## 常见问题

### Q: 我的数据存储在哪里？
A: 所有数据存储在浏览器的 LocalStorage 中。您可以通过浏览器开发者工具查看：
- 打开开发者工具 (F12)
- 选择 "应用" 或 "Application" 标签
- 找到 "LocalStorage" 中的 "project-hub" 条目

### Q: 如果我清除浏览器缓存，数据会丢失吗？
A: 是的，清除浏览器缓存会删除 LocalStorage 数据。建议定期导出数据作为备份。

### Q: 可以在多个浏览器间同步数据吗？
A: 目前不支持自动同步。您可以通过导出/导入功能手动同步数据。

### Q: 支持在手机上使用吗？
A: 是的，应用支持响应式设计，可在手机和平板上使用。

### Q: 如何在离线状态下使用？
A: 应用完全离线工作。只需在有网络时加载一次，之后即可离线使用。

### Q: 数据会自动保存吗？
A: 是的，所有操作都会立即保存到 LocalStorage。

### Q: 可以导出为其他格式吗？
A: 目前支持 JSON 和 CSV 格式。JSON 包含所有信息，CSV 仅包含资源列表。

### Q: 密码是否真的加密了？
A: 是的，使用 AES-256-GCM 加密。您可以查看源代码中的 `lib/encryption.ts` 了解详情。

### Q: 如果忘记了主密码怎么办？
A: 目前无法恢复。建议保存主密码在安全的地方。

### Q: 支持团队协作吗？
A: 目前不支持实时协作。您可以通过导出/导入功能分享项目数据。

### Q: 应用是否收集我的数据？
A: 不收集。应用完全离线工作，无任何数据上传。
