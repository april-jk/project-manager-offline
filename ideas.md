# Project Hub - 设计理念探索

## 设计背景

这是一个离线项目管理工具，用户需要在一个统一的平台上管理多个项目、网站资源和敏感信息。设计需要平衡**专业性**、**易用性**和**视觉吸引力**。

---

## 设计方案 1: 现代极简主义 (Modern Minimalism)

**设计运动**: 极简主义 + 当代科技风格

**核心原则**:
- 极致的留白和清晰的信息层级
- 单一、清爽的配色方案
- 功能优先，装饰最小化
- 微妙的交互反馈

**配色理念**:
- 主色: 深蓝色 (`#1e40af`) - 传达专业性和信任
- 辅助色: 浅灰色 (`#f8fafc`) - 提供视觉呼吸空间
- 强调色: 青绿色 (`#06b6d4`) - 用于成功状态和CTA
- 文本: 深灰色 (`#1e293b`) - 高对比度，易读性强

**布局范式**:
- 左侧固定导航栏 (项目列表)
- 中央主内容区域 (网站黄页或备忘录)
- 右侧可选侧边栏 (项目详情或设置)
- 卡片式布局，间距均匀

**标志性元素**:
1. 圆角卡片 (8px 圆角) - 现代感
2. 细线分割线 - 区分不同内容块
3. 渐进式披露 - 点击展开详情

**交互理念**:
- 平滑的过渡动画 (200-300ms)
- 悬停效果: 卡片轻微上升 + 阴影加深
- 加载状态: 骨架屏或微妙的脉冲动画

**动画指南**:
- 页面进入: 淡入 + 向上滑动 (300ms)
- 卡片交互: 缩放 (1 → 1.02) + 阴影加深
- 模态框: 背景模糊 + 内容缩放进入

**排版系统**:
- 标题: Geist Sans (700) - 现代、几何感强
- 正文: Inter (400) - 高易读性
- 代码: Fira Code (400) - 等宽字体
- 字体大小: 12px (小) → 14px (正文) → 16px (小标题) → 20px (大标题) → 28px (页面标题)

**概率**: 0.08

## 设计方案 2: 温暖有机风格 (Warm Organic Design)

**设计运动**: 新有机主义 + 手工感

**核心原则**:
- 自然的曲线和有机形状
- 温暖的色调，传达亲切感
- 手绘风格的装饰元素
- 柔和的阴影和纹理

**配色理念**:
- 主色: 温暖的琥珀色 (`#d97706`) - 温暖、可信
- 辅助色: 奶油色 (`#fef3c7`) - 柔和的背景
- 强调色: 森林绿 (`#15803d`) - 生机和增长
- 中性色: 暖灰色 (`#78716c`) - 自然感

**布局范式**:
- 非对称布局，左重右轻
- 波浪形分割线分隔不同区域
- 卡片采用不规则边界 (clip-path 创建有机形状)
- 网格布局中混入不同大小的卡片

**标志性元素**:
1. 有机形状的卡片边界 (使用 clip-path)
2. 手绘风格的装饰线条
3. 自然纹理背景 (subtle grain)

**交互理念**:
- 柔和的过渡，强调流畅性
- 悬停效果: 卡片轻微旋转 + 颜色加深
- 加载状态: 脉冲动画，柔和的呼吸感

**动画指南**:
- 页面进入: 淡入 + 缩放 (从 0.95 到 1)
- 卡片交互: 旋转 (±2°) + 颜色变化
- 滚动效果: 视差滚动，卡片随滚动轻微浮动

**排版系统**:
- 标题: Playfair Display (700) - 优雅、有个性
- 正文: Poppins (400) - 温暖、友好
- 代码: JetBrains Mono (400) - 现代等宽字体
- 字体大小: 12px (小) → 15px (正文) → 18px (小标题) → 24px (大标题) → 32px (页面标题)

**概率**: 0.07

## 设计方案 3: 深色科技风格 (Dark Tech Aesthetic)

**设计运动**: 赛博朋克 + 深色模式优先

**核心原则**:
- 深色背景，高对比度
- 霓虹色强调，科技感十足
- 网格和几何图案
- 动态的视觉效果

**配色理念**:
- 背景: 深灰/黑色 (`#0f172a`) - 深邃的科技感
- 主色: 霓虹紫 (`#a855f7`) - 视觉冲击力强
- 辅助色: 霓虹青 (`#06b6d4`) - 对比鲜明
- 文本: 浅灰色 (`#e2e8f0`) - 高对比度

**布局范式**:
- 网格布局，严格的对齐
- 边框使用霓虹色边框线
- 卡片采用玻璃态效果 (backdrop-filter)
- 网格背景图案

**标志性元素**:
1. 霓虹色边框和发光效果
2. 玻璃态卡片 (glassmorphism)
3. 网格背景纹理

**交互理念**:
- 快速、有力的过渡
- 悬停效果: 边框发光 + 背景色变化
- 加载状态: 旋转的网格动画

**动画指南**:
- 页面进入: 扫描线效果 + 淡入
- 卡片交互: 边框发光 + 背景闪烁
- 模态框: 故障效果 + 快速出现

**排版系统**:
- 标题: IBM Plex Mono (700) - 科技感强
- 正文: Space Mono (400) - 未来感
- 代码: Courier Prime (400) - 复古科技
- 字体大小: 11px (小) → 13px (正文) → 16px (小标题) → 22px (大标题) → 30px (页面标题)

**概率**: 0.06

## 选择理由

我选择**方案 1: 现代极简主义**作为最终设计方向。

### 理由:

1. **适配场景**: 项目管理工具需要清晰的信息层级和高效的交互，极简主义完美满足这一需求。

2. **易用性**: 清爽的界面减少认知负担，用户能快速定位和操作功能。

3. **专业性**: 深蓝色和灰色的组合传达专业、可信的感觉，适合开发者和运营人员。

4. **跨平台适配**: 极简设计在各种屏幕尺寸上都能保持良好的视觉效果。

5. **可维护性**: 简洁的设计系统易于扩展和维护，后续添加新功能时不会显得突兀。

### 最终设计决策:

- **主色**: 深蓝色 (`#1e40af`)
- **辅助色**: 浅灰色 (`#f8fafc`)
- **强调色**: 青绿色 (`#06b6d4`)
- **排版**: Geist Sans (标题) + Inter (正文)
- **布局**: 左侧导航 + 中央内容 + 卡片式设计
- **动画**: 平滑过渡 (200-300ms) + 微妙的悬停效果
