# 更新日志

所有重要的项目变更都记录在本文件中。

## [1.0.0] - 2026-01-25

### 新增
- ✨ 项目管理功能
  - 创建、编辑、删除项目
  - 项目彩色标签
  - 项目统计信息

- ✨ 资源管理功能
  - 网站资源（URL + 可选账户信息）
  - API 资源（API Key + Secret + 端点）
  - 备忘录管理

- ✨ 安全特性
  - AES-256-GCM 加密
  - PBKDF2 密钥派生（100,000 次迭代）
  - 本地存储，无服务器上传

- ✨ 用户界面
  - 高密度网格展示
  - 资源预览对话框
  - 密码显示/隐藏切换
  - 快速访问按钮

- ✨ 数据管理
  - 导出数据为 JSON
  - 导入数据恢复
  - 清空所有数据
  - 自动保存

- ✨ 响应式设计
  - 桌面端支持
  - 平板端支持
  - 手机端支持

### 技术栈
- React 19
- TailwindCSS 4
- shadcn/ui
- Vite
- TypeScript
- Web Crypto API
