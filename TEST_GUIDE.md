# Project Hub - 测试指南

## 功能测试清单

### 1. 项目管理

- [ ] **创建项目**
  - 点击"新建项目"按钮
  - 填写项目名称、描述和选择颜色
  - 验证项目已添加到列表中

- [ ] **编辑项目**
  - 点击项目卡片上的编辑按钮
  - 修改项目信息
  - 验证更改已保存

- [ ] **删除项目**
  - 点击项目卡片上的删除按钮
  - 确认删除
  - 验证项目已从列表中移除

### 2. 网站黄页

- [ ] **添加网站**
  - 进入项目详情页
  - 点击"添加网站"按钮
  - 填写网站名称、URL、描述和标签
  - 验证网站已显示在网格中

- [ ] **网站卡片显示**
  - 验证网站图标正确显示
  - 验证网站名称和域名显示
  - 验证标签正确显示

- [ ] **打开网站**
  - 悬停在网站卡片上
  - 点击"打开"按钮
  - 验证网站在新标签页中打开

- [ ] **编辑网站**
  - 点击编辑按钮
  - 修改网站信息
  - 验证更改已保存

- [ ] **删除网站**
  - 点击删除按钮
  - 确认删除
  - 验证网站已从列表中移除

### 3. 备忘录管理

- [ ] **添加备忘录**
  - 进入项目详情页
  - 点击"添加备忘录"按钮
  - 填写标题、内容、分类
  - 验证备忘录已显示在列表中

- [ ] **编辑备忘录**
  - 点击备忘录卡片
  - 修改内容
  - 验证更改已保存

- [ ] **删除备忘录**
  - 点击删除按钮
  - 确认删除
  - 验证备忘录已从列表中移除

### 4. 加密功能

- [ ] **设置密码**
  - 点击菜单中的"设置密码"
  - 输入至少 8 个字符的密码
  - 验证密码已设置

- [ ] **加密备忘录**
  - 创建新备忘录
  - 勾选"🔒 加密存储"选项
  - 验证备忘录显示加密图标

- [ ] **解锁加密内容**
  - 刷新页面
  - 点击"解锁"按钮
  - 输入密码
  - 验证加密备忘录内容可见

### 5. 数据管理

- [ ] **导出数据**
  - 点击菜单中的"导出数据"
  - 验证 JSON 文件已下载
  - 检查文件包含所有项目、网站和备忘录

- [ ] **导入数据**
  - 点击菜单中的"导入数据"
  - 选择之前导出的 JSON 文件
  - 验证数据已导入

- [ ] **清空数据**
  - 点击菜单中的"清空所有数据"
  - 确认操作
  - 验证所有数据已删除

### 6. 响应式设计

- [ ] **桌面版**
  - 在 1920x1080 分辨率下测试
  - 验证布局正确

- [ ] **平板版**
  - 在 768x1024 分辨率下测试
  - 验证布局响应式调整

- [ ] **手机版**
  - 在 375x667 分辨率下测试
  - 验证所有功能可用

### 7. 本地存储验证

- [ ] **数据持久化**
  - 创建项目、网站和备忘录
  - 刷新页面
  - 验证数据仍然存在

- [ ] **浏览器开发者工具**
  - 打开 DevTools → Application → LocalStorage
  - 验证 `pm-projects`、`pm-websites`、`pm-memos` 键存在
  - 验证数据正确序列化为 JSON

## 性能测试

- [ ] **加载时间**
  - 首次加载应该在 2 秒内完成
  - 页面切换应该立即响应

- [ ] **大数据量测试**
  - 创建 50+ 个项目
  - 验证列表仍然可用
  - 验证性能没有明显下降

## 安全测试

- [ ] **密码强度验证**
  - 尝试设置少于 8 个字符的密码
  - 验证系统拒绝

- [ ] **密码验证**
  - 输入错误的密码
  - 验证解锁失败

- [ ] **加密数据验证**
  - 在 DevTools 中查看加密的备忘录
  - 验证内容已加密（不可读）

## 已知限制

- 应用完全依赖浏览器本地存储，不支持跨浏览器同步
- 清空浏览器数据会删除所有应用数据
- 密码无法恢复，请妥善保管

## 故障排除

### 问题：数据未保存
**解决方案**：检查浏览器是否允许 LocalStorage，查看开发者工具中的 Storage 标签

### 问题：密码解锁失败
**解决方案**：确保输入的密码与设置时完全相同（区分大小写）

### 问题：网站图标不显示
**解决方案**：这是正常的，某些网站可能不支持 Favicon API，会显示默认的地球图标
